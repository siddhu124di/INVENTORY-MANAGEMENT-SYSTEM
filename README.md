# Inventory Management System

## Project Overview
The Inventory Management System is a data-driven application developed using Python, MySQL, and Streamlit. It helps businesses manage inventory efficiently by tracking products, suppliers, stock levels, sales, and reorder requirements.

The system provides real-time inventory insights, automated stock monitoring, and analytical reports to support better business decisions.

---

## Features

### Inventory Management
- Add new products
- Update product information
- Delete products
- View complete inventory

### Stock Monitoring
- Track current stock levels
- Identify low-stock products
- Monitor inventory availability

### Supplier Management
- Store supplier information
- View supplier details
- Track supplier-product relationships

### Sales Analysis
- View sales records
- Analyze product performance
- Monitor inventory movement

### Reorder Management
- Automatic low-stock detection
- Generate reorder requests
- Track reorder status

### Dashboard & Analytics
- Total products
- Total suppliers
- Current stock overview
- Low stock alerts
- Sales insights
